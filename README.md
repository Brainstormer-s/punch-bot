# punch-bot 
punch-bot  is a telegram bot to see the top ten Box Office Movies 

# instalation 
- open **CMD**
- git clone https://github.com/Brainstormer-s/punch-bot.git
- cd punch-bot
- pip install urllib
- pip install telegram-python-bot
- pip instal bs4
- python app.py

Don't Forget To Change The Bot Token **!**
